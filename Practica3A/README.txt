	Para ejecutar la practica solo es necesario situarse en la carpeta dist y ejecutar el siguiente comando
		
	java -jar pracale3.jar


	Para meter los argumentos es necesario poner los argumentos como viene:
	
	<Ruta del Archivo a leer><Numero que queremos buscar><Busqueda>
	
	Para la busqueda se encuentran las siguientes opciones:
	
	Binaria
	Exponencial
	Secuencial
	Interpolacion

	Para ver las listas generadas adentro en el codigo viene una clase llamada genere listas.
